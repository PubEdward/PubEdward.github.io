开源工具池

  1. 财会类软件

    • Edoceo Imperium

      专为中小型企业（SMB）开发的 Edoceo Imperium 将会计、发票和基本的客户关系管理（CRM）功能结合并集成了谷歌的日历，地图和电子邮件（Gmail）功能，支持多达四十亿个不同访问级别的用户。

      支持的操作系统：无限制

    • FrontAccounting

      同样为中小企业设计的 FrontAccounting 包括一些基本的企业资源规划（ERP）功能，如库存和生产跟踪。该公司网站提供帮助潜在用户体验其产品的演示。

      支持的操作系统：无限制

    • TurboCASH

      这一会计软件包旨在提供大多数中小企业所需的全部功能包括债务，债权，总帐，库存管理，发票， 调账，试算表和报告等。支持多公司和多用户。

      支持的操作系统：Windows

    • XIWA

      仅在 Linux 环境运行的 XIWA 意为 ” XIWA 是网络会计”。 自1999年开发至今，它具有支持多用户和多公司，工资，股票/投资跟踪，报表等功能。

      支持的操作系统：Linux

  2. 录音/编辑类软件

    • MuseScore

      乐谱软件通常花费数百美元。但 MuseScore 创作优美的乐谱却是完全免费的。其特点包括无限和弦长度，无限制谱线数量，无限的声音输入数量，创作向导和模板，自动部分提取，换位，力度，歌词，和弦符 号，打击乐谱符号，早期乐谱符号等等。

      支持的操作系统：Windows，Linux 和 OS X

    • Ardour

      为满足专业人士的需要而开发，Ardour 提供全套录音，编辑和混音工具。主要特点包括无限量音轨和数据总线，支持破坏性/非破坏性编辑录音模式，支持数十种文件格式，“业内最灵活的混音器架 构”，数以百计的插件，精确自动化采样，视频时间轴和更多的支持。

      支持的操作系统：Linux，OS X

    • Audacity

      一个易于使用的录音和编辑应用程序，对于业余音乐人和家庭用户是一个很好的选择。它可将磁带，唱片和光盘转换成数字文件，并支持 WAV，AIFF，FLAC，MP2，MP3 或 Ogg Vorbis 文件格式。

      支持的操作系统：Windows，Linux 和 OS X

    • Frinika

      自称为一个“完整的音乐工作站”。它的用户界面和功能的先进性尚未达到一些专业人士的要求，但它提供音序控制，MIDI 接口支持, 软件合成，录音和乐谱编辑等功能。

      支持的操作系统：无限制

    • Linux MultiMedia Studio

      (LMMS) 包括歌曲编辑，节拍和贝斯编辑，钢琴卷帘窗，64通道 FX 调音和更多功能。最初为 Linux 开发，现有 Windows 版本。

      支持的操作系统：Windows 和 Linux

    • WaveShop

      快速和小巧。值得关注的特点包括调音，淡化，采样率转换，音频生成，无限量撤销，广泛的帮助，峰值，方均根（RMS）和频谱分析。

      支持的操作系统：Windows

  3. 商业智能类软件

    • JasperSoft

      作为最流行的开源商业智能工具之一，JasperSoft 提供可视化企业信息仪表板，自助报表，丰富的分析，移动支持等更多功能。该公司还在 Jaspersoft.com 提供付费的商业版本。

      支持的操作系统：无限制

      • Jedox

        原名 Palo BI 的 Jedox 是一个完整的商业智能套件，包括一个联机分析处理(OLAP)服务器，Excel 插件，数据提取，转换和加载（ETL）和移动支持。基础版本只包括联机分析处理(OLAP)服务器和有限版 Excel 插件，可在开源许可证下免费使用。企业內部部署和云版本需付费。

        支持的操作系统：无限制

    • Pentaho

      提供数据集成，业务分析和大数据工具。开源社区版本可从 community.pentaho.com 下载。 用户包括汉莎航空，哈里伯顿，纳斯达克和江户国际等企业。

      支持的操作系统：Windows，Linux 和 OS X

    • Spago

      声称是“唯一100％开源的，完整和灵活的商业智能套件。”它建立在其它开源项目的广泛基础上，提供模块化的报表，联机分析处理(OLAP)，图表，关键 性能指标（KPI），互动管理驾驶舱，自助报表，位置智能，数据挖掘，信息仪表板，团队协作，数据提取，转换和加载（ETL），主数据管理等等。

      支持的操作系统：无限制

    • KNIME

      用户遍及60多个国家，KNIME 自称是“领先的数据驱动创新开放平台帮助企业引领变革。”其核心分析软件为开源免费，商业扩展则为有偿服务。

      支持的操作系统：Windows，Linux 和 OS X

    • OpenI

      这一插件扩展了 Pentaho 和 Jasper 的分析工具能力承诺让新用户在短短72小时内运行起来。它包括信息仪表板，交互式报表，数据提取，转换和加载（ETL），维度模型 （dimensional modeling），多维联机分析处理(OLAP)设计和预测建模。

      支持的操作系统：无限制

    • BIRT

      功能范围略逊于清单上的其它软件，BIRT 专注于报表创建和可视化嵌入式应用。它的赞助公司为Actuate（提供各种相关的资源），Innovent Solutions 和 IBM。需要注意的是，你需要在 Eclipse 开发环境下使用它。

      支持的操作系统：Windows，Linux 和 OS X

  4. 业务流程管理类软件

    • ProcessMaker

      自称为开源工作流和业务流程管理（BPM）软件”提供了工作流程建模和运行，报表创建和工作流程优化工具。 除了开源版本，它还有基于企业內部部署和云的付费版本。

      支持的操作系统：Windows 和 Linux

    • Activiti

      基于Java的 Activiti 的特点为尺寸小，运行快。Alfresco 公司可为它提供付费支持。

      支持的操作系统：无限制

    • Bonita

      是一个完整业务流程管理（BPM）套件，包括组织结构图，数据结构定义，用户界面构建，可执行报表创建等众多工具。除了免费的社区版，它的 “团队合作”，“提高效率”和“能力拓展“付费版本可提供相应的增强功能。

      支持的操作系统：无限制

  5. 计算机辅助设计类软件

    • FreeCAD

      易于使用的三维参数化建模工具, 专为家庭用户和发烧友设计。其模块化架构十分便于定制和扩展。

      支持的操作系统：Windows，Linux 和 OS X

    • Archimedes

      基于 Eclipse 的富客户端平台（Rich Client Platform），每周下载量达数百次的阿基米德(Archimedes)提供了最小功能集的计算机辅助设计（CAD）简化解决方案。

      支持的操作系统：Windows，Linux 和 OS X

  6. 客户关系管理类软件

    • Sugar Community Edition

      世界各地数以百万计的用户使用 Sugar 提供的工具将销售，营销和支持团队与潜在客户和客户联系到一起。付费的云版本可以在 SugarCRM.com 找到。

      支持的操作系统：无限制

    • vTiger Replaces Sage ACT!

      超过10,000个组织在使用这个客户关系管理（CRM）套件。其功能包括电子邮件营销，项目管理，客户门户，日历，任务管理，报告，文件管理，访问控制，库存管理等。其云版本为付费版本。

      支持的操作系统：Windows，Linux，iOS， Android

    • Orange Leap Replaces Sage ACT!

      原名 MpowerOpen 的 Orange Leap 提供社团成员关系管理帮助非营利组织开展筹款活动。源代码可从 SourceForge 下载。

      支持的操作系统：Windows。

    • SourceTap

      旨在帮助销售团队改善预测准确度，提高交易成功率，缩短销售周期，增强协作和辨识市场趋势。它可提供商业或开源许可证。

      支持的操作系统：Windows 和 Linux。

    • SplendidCRM

      有多个基于企业內部部署，云，付费和免费的版本。最新版本更好地集成了 Twitter和 Twilio 短信通讯（SMS），反删除能力，定时重复会议，HTML5 日历等。

      支持的操作系统：Windows

    • ConcourseSuite

      基于 Java 的 ConcourseSuite 整合了销售，营销和客户服务团队使用的工具。除了免费版本，它还有付费的云和企业內部部署版本。

      支持的操作系统：Windows，Linux 和 OS X

  7. 桌面出版类软件

    • Scribus

      适合专业平面设计师和出版商使用的 Scribus 是一个全功能的平面设计软件。支持的功能包括分色，CMYK 和专色，ICC 色彩管理，创建 PDF 文档等等。

      支持的操作系统：Windows，Linux 和 OS X

    • MiKTeX

      如果您要创建有很多科学表达式和数学公式的文件，请考虑 MIKTEX。它是专门“为创作美好的书，特别是包含大量数学的书籍”设计的。

      支持的操作系统：Windows 和 Linux

  8. 开发工具类软件

    • KDevelop

      这个从 KDE 发展而来的集成开发环境（IDE）支持 C / C++ 和其它编程语言。自1998年开发以来，它已成为一个成熟，稳定，有着庞大用户群的项目。

      支持的操作系统：Linux，Unix，OS X

    • LiteIDE

      LiteIDE是一款开源、跨平台的轻量级Go语言集成开发环境（IDE）。

    • MonoDevelop

      专为 C＃ 和 Visual Basic.NET 设计，MonoDevelop 帮助 Windows 开发人员创建多平台应用程序。主要功能包括先进的文本编辑，集成调试器，GTK＃ 可视化设计，单元测试，本地化功能等等。

      支持的操作系统：Windows，Linux 和 OS X

    • SharpDevelop

      .NET 开发人员的另一个很好的选择，SharpDevelop 的（#develop）支持 C＃，VisualBasic.NET 和 Boo。它包括一个集成的调试器，代码分析，单元测试，代码覆盖率，特征剖析，整合的 Git 和 Subversion 支持等等。

      支持的操作系统：Windows

    • OpenBD

      原名 Open Blue Dragon 的 Open BD 使不购买 Adobe Cold Fusion 就能开发和部署基于 Cold Fusion 标记语言（CFML）的网络应用成为可能。它自冠的称号为：“世界上第一个真正的开源和免费的基于 GPL 和 Java 的 CFML 运行时环境。”

      支持的操作系统：Windows 和 Linux

  9. 电子邮件/协同/群件类软件

    • Zimbra

      Zimbra 的开源版本提供了一个类似 Microsoft Exchange 功能的跨平台的客户端到服务器端邮件解决方案。该公司还提供拥有5亿用户的多种付费通信应用程序。

      支持的操作系统：Windows，Linux，Unix，OS X

    • EGroupware

      据 eGroupWare 的首席执行官讲，基于网络的 eGroupWare 软件是“领先的开源协作工具，为全球大，中小企业的内部和跨组织团队的首选。”社区版可免费在内部服务器上运行。 企业版有定期付费的云版本或一次性付费的安装版。

      支持的操作系统：无限制

    • Citadel

      Citadel 自封“真正的开源电子邮件和协作软件的领导者”，并承诺让用户“在几分钟内运行起来”无论是采用自己的硬件或云计算服务。其功能包括电子邮件，日历，论 坛，邮件列表服务器，即时通信（IM），维基，博客，简易信息聚合(RSS)的整合等。

      支持的操作系统：Linux

  10. 企业资源规划（ERP）

    • Odoo

      原名 OpenERP 的 Odoo 自称捆绑了多种应用程序如网站建设，促销，业务经营，市场营销，提高员工满意度和生产效率。您可以免费下载该软件，或者使用有偿云版本每应用程序每用户每月15美元。

      支持的操作系统：Windows，Linux 和 OS X

    • Openbravo

      Openbravo 提供了一个电子商务和企业资源规划（ERP）平台，用户包括 Oracle，江森自控，联邦银行，Grupo Eulen 和其它公司。开源版本可在 Code.Openbravo.com 下载。您还可以通过Openbravo 的渠道合作伙伴购买企业版并获得支持。

      支持的操作系统：无限制

    • Postbooks/xTuple ERP

      宣称自己是“世界第一开源企业资源规划（ERP）软件”。xTuple 提供多种版本包括专门针对分销商和制造商的应用。PostBooks 有开源和商业版。多个应用程序和扩展可以通过 xChange 在线市场获得。

      支持的操作系统：无限制

    • ADempiere

      这个具有社区意识的 ERP 解决方案包括销售，采购，生产管理，物料管理，财务，人力资源，项目管理，客户关系管理（CRM），销售点终端（POS），电子商务，设备维护管理和报表 功能。该软件基于 Compiere 代码（见下一条目），无商业支持，但该网站提供了详尽的技术文档和培训资料。

      支持的操作系统：Windows，Linux，OS X

    • Compiere ERP + CRM Business Solution

      自称是“最现代化，适应性强和用得起的企业资源规划（ERP）解决方案。”它的功能类似于 Adempiere。它提供了付费支持的企业版和在亚马逊网络（Amazon Web Services）上运行的选项。

      支持的操作系统：Windows，Linux，OS X

    • Opentaps

      Opentaps 的用户包括丰田，霍尼韦尔，mBlox 等。它将企业资源规划（ERP）和客户关系管理（CRM）功能结合并提供有偿服务和支持。亚马逊网络服务（Amazon Web Services）也可提供Opentaps 服务。

      支持的操作系统：Windows 和 Linux

    • Apache OFBiz

      Apache OFBiz 是"Open for Business"的缩写。它提供了一套完整的企业自动化工具，包括电子商务，目录管理，推广和定价管理，订货，客户管理，会计，内容管理，销售点终端（POS）等。它的设计理念是灵活兼易于订制。

      支持的操作系统：无限制

    • Dolibarr ERP/CRM

      Dolibarr ERP/CRM 是为小公司，基金会和自由职业者专门设计的 Dolibarr 可作为一个独立的应用程序或云服务使用。相关的很多插件可在 Dolistore 网站获得。

      支持的操作系统：无限制

    • WebERP

      WebERP 承诺提供成熟实用的基于网络的业务运行软件，特别是针对批发商，分销商和制造商的业务需求。分布在世界各地的多个第三方公司可提供付费支持。

      支持的操作系统：无限制

  11. Foreign Language 外语类软件

    • Step Into Chinese

      虽然它没有罗塞塔石碑(Rosetta Stone Chinese)那样完整的培训计划，Step Into Chinese 提供了多种中文自学工具，包括抽认卡片和超过26,000个中文词和8,300个汉字的字典。

      支持的操作系统：Windows，Linux 和 OS X

    • ZWDisplay

      学习中文的另一个好工具，ZWDisplay 强调阅读和听力技巧。它采用了特殊的颜色编码格式，帮助用户学习中文的音调。

      支持的操作系统：Linux

    • Zkanji

      Zkanji 的字典收录了21万个日文单词以帮助用户读写日文并结合抽认卡片练习词汇量。 它的目的是为初学者和高级学员提供帮助。

      支持的操作系统：Windows

  12. 网关安全设备

    • Endian Firewall Community

      用户可以利用 Endian 的社区版和任何x86标准硬件创建自己的统一威胁管理（UTM）设备。Endian 还销售硬件，软件和基于同一技术的虚拟设备。

      支持的操作系统：Linux

    • Untangle

      Untangle 的免费 NG 防火墙包括网络过滤，应用控制，防病毒，防网络钓鱼，防入侵，防火墙，OpenVPN，防垃圾邮件，广告拦截和报表生成等开源解决方案。同 Endian 一样，它支持用户利用旧电脑创建自己的安全设备，用户也可购买该系统支持的统一威胁管理（UTM）软件或硬件。

      支持的操作系统：Windows 和 Linux

  13. 图形/绘图类软件

    • Dia

      同 Visio 一样，Dia可以创建关系图，流程图，网络图，组织结构图和类似的图表。在默认情况下它支持 XML 文件，可以导出 EPS，SVG，XFIG，WMF 和 PNG 文件格式。

      支持的操作系统：Windows 和 Linux

    • Gimp

      Gimp 是“GNU图像处理程序”的简称，它提供专业品质的照片编辑和图像制作，具有与 Photoshop 相同的高级功能，在 Linux 桌面用户中备受青睐。

      支持的操作系统：Windows 和 Linux

    • Inkscape

      Inkscape 适合专业平面设计师使用。它提供了创建矢量图形和支持多种不同文件类型的各种工具。

      支持的操作系统：Windows，Linux 和 OS X

  14. 数学类软件

    • Sage

      Sage 为超过100种不同的开源数学项目提供了一个统一的界面。它可以解析基础代数，微积分，数论，密码学，交换代数，群论，2D 和 3D 图形，线性图（linear graphics）等等。

      支持的操作系统：Windows，Linux 和 OS X

    • Genius

      Genius 可处理多种类型数学运算，包括微积分，数论，统计，三角函数，模运算，2D 和 3D 绘图等等。它包括一个图形用户界面的开发环境（IDE），并可导出 LaTex、Troff 或 MathML 格式矩阵数据。

      支持的操作系统：Linux，OS X

    • Scilab

      许多教育机构使用 Scilab 辅助教授数学，科学和工程课程。它可以执行多种不同类型计算。Scilab 的企业版可提供有偿的支持，培训和服务。

      支持的操作系统：Windows，Linux 和 OS X

    • Maxima

      计算机代数系统可求解的数学表达式包括“微分，积分，泰勒级数，拉普拉斯变换，常微分方程，线性方程组，多项式，以及集合，数列，向量，矩阵和张量”等。 它的代码基于 MIT 在20世纪60年代末开发的 MACSYMA 数学程序。

      支持的操作系统：Windows，Linux 和 OS X.

  15. 办公效率

    • OpenOffice

      完整版 Microsoft Office 的价格近年来显著回落，但使用年费还是随着时间的推移而显著累积。相比之下，完整版本的 Apache OpenOffice 是完全免费的。它包括文字处理，电子表格，演示文稿，图片，数学公式编辑和数据库软件，它还可读写 Office 格式文件。

      支持的操作系统：Windows，Linux 和 OS X

    • LibreOffice

      从 OpenOffice 分支出来的 LibreOffice 提供其前身相同类型的软件并附带额外的功能扩展。其网站上的对比表有助于比较它与 Microsoft Office 的异同（注：在官方网站未找到此表。但在 Wiki 网可找到https://wiki.documentfoundation.org /FeatureComparison:_LibreOffice-_Microsoft_Office）。

      支持的操作系 统：Windows，Linux 和 OS X

    • Calligra

      Calligra 前身为 KOffice，Calligra 套件包括文字处理器，电子表格，演示文稿，项目管理，绘画（Krita），数据库（Kexi），矢量图形（Karbon），流程图和思维导图。它支持微软 的文件格式，但用户界面与 Office 很不同。

      支持的操作系统：Windows，Linux 和 OS X

    • NeoOffice

      NeoOffice 也是从 OpenOffice 分支出来的。但 NeoOffice 仅用于 Mac 电脑。最新版本支持 Yosemite 系统的新 iCloud 驱动，可从 Mac App Store 下载。

      支持的操作系统：OS X，iOS

  16. 销售点终端（POS）

    • Openbravo POS

      这个基于网络的销售点终端（POS）解决方案集成了 Openbravo 的企业资源规划（ERP）软件。移动版可以让零售人员帮助客户在店内任何地方结算。

      支持的操作系统：Windows，Linux 和 OS X

    • uniCenta

      与 Openbravo POS 基于相同源代码, 是一个多语种的销售点终端（POS）系统十分适合在多功能设施使用，如带零售店的餐厅。它已被下载超过13万次。

      支持的操作系统：Windows，Linux，OS X

    • Floreant POS

      仅适于餐馆，Floreant 已被用在超过25,000个场所，包括一些丹尼（Denny's）连锁餐厅。它可处理订货，厨房打印，钱屉，外卖，税收，打折，订单分组等等。

      支持的操作系统：Linux
